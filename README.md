#  Codeforces Analyzer

A simple web app that analyzes a user's Codeforces profile and displays useful insights like problems solved, contest participation, rating, and tag-wise breakdown.

---

##  Features

-  Total problems solved (unique)
-  Contest participation count
-  Current & max rating
-  Tag-wise problem analysis (DP, Greedy, Math, etc.)
-  Fast and simple UI

---

##  Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js, Express  
- API: Codeforces Public API  

---


##  How to Run Locally

```bash
git clone https://github.com/your-username/cf-analyzer.git
cd cf-analyzer
npm install
node server.js
